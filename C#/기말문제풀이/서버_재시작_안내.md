# 서버 재시작 안내

## 문제 목록이 보이지 않을 때

### 1. 서버 재시작

서버를 중지하고 다시 시작하세요:

**PowerShell 또는 명령 프롬프트에서:**
```bash
cd "C#\기말문제풀이\server"
npm start
```

### 2. 브라우저 캐시 삭제

브라우저에서:
- `Ctrl + Shift + R` (강력 새로고침)
- 또는 개발자 도구 (F12) → Network 탭 → "Disable cache" 체크

### 3. 문제 데이터 확인

문제 데이터 파일이 정상적으로 존재하는지 확인:
- `server/csharp-sources/problems.json` ✅
- `server/parenting-sources/problems.json` ✅

### 4. 서버 로그 확인

서버 콘솔에서 다음 메시지 확인:
- "서버가 포트 5000에서 실행 중입니다"
- 에러 메시지가 없는지 확인

---

## 문제가 계속되면

1. 서버를 완전히 종료 (Ctrl + C)
2. 잠시 대기 (5초)
3. 다시 시작
